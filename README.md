# Country Info App

A Flutter application that fetches and displays country details using a REST API.

# Features
- List of countries with details:
    - Name
    - Capital
    - Region
    - Population
    - Flag

# Tech Stack
- Flutter
- Dart
- HTTP

# API
https://restcountries.com/v3.1/all
